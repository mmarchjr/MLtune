# Java Integration

Files for integrating MLtune with your robot code.

## Files

- `TunerInterface.java` - NetworkTables interface
- `LoggedTunableNumber.java` - Tunable parameter wrapper
- `FiringSolutionSolver.java` - Example solver implementation
- `Constants_Addition.java` - Example constants

## Usage

Copy `TunerInterface.java` and `LoggedTunableNumber.java` into your robot project.

See [../INTEGRATION.md](../INTEGRATION.md) for integration details.
