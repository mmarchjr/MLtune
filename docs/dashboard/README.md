# MLtune Dashboard

Web-based monitoring and control interface.

## Access

Once running, open http://localhost:8050 in your browser.

## What's Here

- `app.py` - Main dashboard application (Dash/Plotly)
- `assets/` - CSS and JavaScript files
- `requirements.txt` - Dashboard dependencies

## Features

- Real-time tuning status
- Coefficient visualization
- Manual optimization controls
- Shot history and logs
- Configuration toggles

See [../USAGE.md](../USAGE.md) for usage information.
